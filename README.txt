To run the file , on the command line
Assessment.py [task1/task2]

By default, if no parameter is sent, the code for Task-1 is executed.
Else, task1 -> executes code for Task-1
task2 -> executes code for Task-2

The code would generate output on the console as well as in a json file.
Task-1 output -> key_metric_output.json
Task-2 output -> products_output.json

The description and detailed analysis for Task-3 is present in the file Task-3.txt

Please Note: For simplicity, please place the input json files in the same directory as the python file.
For simplicity, the file name is kept the same in the code as they were sent.
